# Quick Start — 5 Minutes to Warp Basics

Get up and running with Warp in just a few minutes.

---

## Installation (2 min)

**macOS:**
```bash
brew install warp
```

**Linux:**
```bash
curl https://releases.warp.dev/releases/stable/warp-v0.2024.01.09/warp-terminal-v0.2024.01.09.AppImage -o warp.AppImage
chmod +x warp.AppImage
./warp.AppImage
```

**Windows:**
Download from [releases.warp.dev](https://releases.warp.dev)

---

## Your First AI Interaction (1 min)

1. **Open Warp** — Launch the app
2. **Type a command** — e.g., `ls -la`
3. **Press Cmd+K** (Mac) or **Ctrl+K** (Windows/Linux) — Ask Warp for help
4. **Ask anything** — "Show me files modified in the last day" or "List all Python files"
5. **Warp runs it** — Review and press Enter to execute

That's it! You're using AI in your terminal.

---

## What Can You Do?

- **Run commands** — "Create a backup of my config files"
- **Explain code** — "What does this grep command do?"
- **Generate scripts** — "Write a script that syncs my dotfiles"
- **Agent mode** — Press Cmd+Shift+K to let Warp autonomously complete tasks

---

## Changing AI Models

By default, Warp uses its recommended model. Want something different?

1. Click **⚙️ Settings** → **AI Models**
2. Choose your model:
   - **Claude 3.5 Sonnet** (best for reasoning)
   - **GPT-4o** (general purpose)
   - **Gemini 2.0** (fast and capable)
   - **Custom** (bring your own API key)
3. Done! Your choice is saved.

👉 See [Selecting Models](./AI_MODELS/SELECTING_MODELS.md) for more details.
