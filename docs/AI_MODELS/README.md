# AI Models in Warp

Warp is powered by AI, and you have full control over which model you use.

---

## What Are AI Models?

AI models are the "brain" behind Warp's intelligent features. Different models have different strengths:

- **Reasoning** — Claude excels at complex problem-solving
- **Speed** — Gemini is fast and responsive
- **Code** — GPT-4o is excellent at code generation
- **Versatility** — Different models for different tasks

You can use Warp's recommended models, or **bring your own API key** and use your preferred model.

## Overview of Model Selection

By default, Warp automatically selects the best model for your use case:
- **Agent Mode** — Optimized for autonomous task completion
- **Code Editing** — Optimized for code suggestions
- **CLI Agent** — Optimized for command line help
- **Computer Use** — Optimized for automation (when available)

You can override any of these in **Settings → AI Models**.
